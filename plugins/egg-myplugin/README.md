# egg-myplugin

A custom plugin for Egg.js

## Installation

```bash
npm install egg-myplugin --save
```

## Usage

```js
// config/plugin.js
exports.myplugin = {
  enable: true,
  package: "egg-myplugin",
};
```

## Configuration

```js
// config/config.default.js
exports.myplugin = {
  // your configuration
};
```
